## 預想情境

K8S 是一各國宅型社區(很多棟)的管理者。每棟國宅(Node)就像(實體機、虛擬機)。國宅中的一戶就是一個(Pod)，每戶有多個人(container)，每個 Pod 是一個對外的最小單位。當其中一棟國宅(Node)太過擁擠時，K8S 會負責把過多的戶(Pod)搬往相對空的大樓。
